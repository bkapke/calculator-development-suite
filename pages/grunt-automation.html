Automation

This app is a demonstration of using grunt tasks in the development workflow. Using grunt you can automate common development tasks to improve the quality of your code, the stability of your app and the speed at which you develop. This demo includes the following common tasks:

  • JavaScript Unit Testing via qUnit
  • Hinting your JavaScript via grunt-contrib-jshint
  • Watching your JS and running tasks on save with grunt-contrib-watch
  • Concatenting multiple JS files and classes into a single file with grunt-contrib-concat
  • Uglifying/Minifying your JS to decrease file size with grunt-contrib-uglify
